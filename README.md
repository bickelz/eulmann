# EULMANN

Static artist site for [eulmann.com](https://eulmann.com).

## Local preview

Open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Deploy

**Vercel:** Connect this repo or run `vercel` from the project root. No build step required.

**Netlify:** Drag-drop the folder or connect the repo. Publish directory: `.` (root).
